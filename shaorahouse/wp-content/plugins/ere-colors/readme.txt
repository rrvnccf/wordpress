== Description ==
Love the Essential Real Estate plugin, but need its colors to match your website? Want to change the color of your site without technical overhead? ERE Colors is the most efficient way to re-color your real estate website. It provides 3 color pickers: Accent Color, Contrast Color and Text Color

== Installation ==

Do follow to install plugins:

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to menu Essential Real Estate -> Real Estate Options/Colors screen to configure the colors of Essential Real Estate plugin
